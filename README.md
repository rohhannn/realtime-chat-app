# 💬 ChatSphere – Real-Time Chat Application

A modern, full-stack real-time chat application built using **Node.js, Express, and Socket.IO**, featuring authentication, live messaging, and a professional UI.

---

## 🚀 Live Demo

🌐 **Live App:** https://chatsphere-app-phqm.onrender.com
📦 **GitHub Repository:** https://github.com/rohhannn/realtime-chat-app

---

## ✨ Features

* 🔐 User Authentication (Signup & Login)
* 💬 Real-time messaging using Socket.IO
* 👥 Live online users tracking
* ⌨️ Typing indicator
* 🔔 Message notifications
* 🔍 Search users functionality
* 🧑‍💻 User avatars & chat bubbles
* 🎨 Modern glassmorphism UI design
* ⚡ Instant message updates (no refresh)

---

## 🛠️ Tech Stack

### Frontend

* HTML5
* CSS3 (Glass UI + Responsive Design)
* JavaScript (Vanilla JS)

### Backend

* Node.js
* Express.js
* Socket.IO (WebSockets)

### Authentication

* Express Session
* Bcrypt (Password hashing)

## ▶️ Run Locally

### 1. Clone repository

```
git clone https://github.com/rohhannn/realtime-chat-app.git
cd realtime-chat-app
```

### 2. Install dependencies

```
npm install
```

### 3. Start server

```
node server.js
```

### 4. Open in browser

```
http://localhost:4000
```

---

## 🧪 How It Works

1. User signs up or logs in
2. Session is created using Express-session
3. Socket.IO establishes real-time connection
4. Users join chat room
5. Messages are broadcast instantly to all users
